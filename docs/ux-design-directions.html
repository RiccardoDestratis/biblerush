Design Direction Notes

All directions feature game-focused 3D styling with gradients, shadows, and depth effects. Buttons have tactile feel with hover/active states. Cards have elevation. This is a game experience, not SaaS.

üí° Key Principle: Every interactive element should feel like it's part of a game - with depth, elevation, and playful interactions. Think game UI, not business application.
